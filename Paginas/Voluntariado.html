    ¿Quiéres ser voluntario?

    Paseo a ciegas es un proyecto ciudadano que funciona gracias a gente que participa de forma desinteresada y con la creencia férrea de que este puede ser día con día un mejor lugar para vivir con respeto y tolerancia. Para poder ser parte de este equipo de voluntarios es necesario: participar en nuestro taller de sensibilización que se imparte el primer sábado de cada mes, acercarte a las diferentes actividades que se realizan en este programa, tener muchas ganas de ayudar y de conocer gente con un punto de vista diferente.



    Taller de sensibilización


    Para generar conciencia ciudadana llevamos a cabo la realización de un taller de sensibilización el primer sábado de cada mes; con el fin de crear una empatía hacia las personas con discapacidad.

    ¿Cuándo se lleva a cabo?


    El taller se lleva a cabo el primer sábado de cada mes, de 10:00 a 14:00 hrs. Para poder participar se requiere reservación con una semana de anticipación.
